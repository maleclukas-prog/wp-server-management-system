# 🚀 WordPress Server Management System (WSMS)

A production-grade suite of automation tools for managing high-performance, multi-tenant WordPress environments on Ubuntu Server.

## 🌟 Overview
This system was developed to solve real-world operational challenges in managing secure, scalable, and automated web infrastructures. It bridges the gap between server administration and application maintenance by providing a unified automation layer.

## 🛠 Key Features
- **Smart Retention Engine:** Proactive disk space management with "Last-Copy-Safe" logic.
- **Hybrid Cloud Sync:** Automated off-site synchronization to Synology NAS via secure SFTP.
- **Multi-Tenant Security:** Full infrastructure hardening using system-user isolation and granular ACLs.
- **Fleet Observability:** Real-time health audits and status monitoring for all hosted instances.
- **Disaster Recovery:** Multi-tier backup strategy (Lite Assets, Full Snapshots, and Bare-metal Recovery).

## 💻 Tech Stack
- **Languages:** Bash (Shell Scripting)
- **Integrations:** WP-CLI, MySQL, ClamAV, SFTP/SSH
- **OS:** Ubuntu Server (Optimized for 22.04/24.04 LTS)

## 📖 Operational Mindset
This project demonstrates an **Ownership** and **Security-First** approach. By automating 95% of routine maintenance (patching, backups, rotation), it ensures high system availability and minimizes technical debt.
